# 🏥 QueueZero

A Smart Queue Management System integrated with a Hospital Management System (HMS) to reduce patient waiting time and improve hospital workflow efficiency.

---

## ✨ Key Highlights

* Real-time patient token generation
* Doctor-side queue control system
* QR-based token tracking
* SMS notifications using Twilio
* Efficient queue optimization for hospitals

---

## 🚀 Features

* Token Generation System
* Real-time Queue Updates
* Doctor Dashboard to Call Next Patient
* Patient Token Tracking
* QR Code Integration
* SMS Notification (Twilio)

## ⚙️ Installation

```bash
git clone https://github.com/snehabenny12/QueueZero.git
cd QueueZero

# Create virtual environment
python -m venv venv
venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt

# Create .env file and add:
# SECRET_KEY=your_secret_key
# TWILIO_ACCOUNT_SID=your_sid
# TWILIO_AUTH_TOKEN=your_token

python manage.py runserver
```
