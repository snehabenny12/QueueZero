# QueueZero 🏥

A Smart Queue Management System integrated with Hospital Management System (HMS) to reduce patient waiting time and improve hospital efficiency.

## 🚀 Features
- Token Generation System
- Real-time Queue Updates
- Doctor Dashboard to Call Next Patient
- Patient Token Tracking
- QR Code Integration (optional)
- SMS Notification (Twilio)

## 🛠 Tech Stack
- Python (Django)
- SQLite / MySQL
- HTML, CSS, JavaScript
- Bootstrap

## 📊 Modules
- Patient Registration
- Token Management
- Queue Monitoring
- Doctor Panel
- Admin Panel
